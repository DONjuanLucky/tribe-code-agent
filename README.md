# Tribe Code Agent (TCA) - Powered by OpenClaw

TCA is a self-hosted AI agent overlay for the Tribe Code ecosystem, built on top of [OpenClaw](https://github.com/openclaw/openclaw).

This repository contains:
1. **Workspace Files**: The agent's identity, soul, and operational behavior configuration.
2. **Tribe UI Overlay**: A custom, modern interface for the OpenClaw Control panel with Khaki editorial theming and new chat capabilities.
3. **Advanced Skills**: Pre-packaged skills for software architecture, coding, UI/UX, and more.

## Quick Start

1. Install Node.js 22.
2. Install OpenClaw globally: `npm install -g openclaw@latest`
3. Point your OpenClaw workspace to this repository:
   Make sure `~/.openclaw/openclaw.json` (or your local `openclaw.json`) points its workspace directory to `TRIBECODEOPENCLAW/workspace`.

For step-by-step instructions, see [SETUP.md](./SETUP.md).
